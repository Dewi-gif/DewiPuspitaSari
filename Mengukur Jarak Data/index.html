    Mengukur Jarak Data¶

    Mengukur Jarak Tipe Numerik¶

    Minkowski Distance¶

    Jarak Minkowski adalah metrik dalam ruang vektor normal ang dapat dianggap sebagai generalisasi dari jarak Euclidean dan jarak Manhattah.

    d _ { \operatorname { min } } = ( \ sum _ { i = 1 } ^ { n } | x _ { i } - y _ { i } | ^ { m } ) ^ { \frac { 1 } { m } } , m \geq 1

    Manhattan distance¶

    Manhattan adalah manhattan yang tersusun menjadi blok-blok. sehingga sering juga disebut city block distance, juga sering disebut sebagai ablosute value distance atau boxcar distance.

    d _ { \operatorname { man } } = \sum _ { i = 1 } ^ { n } \left| x _ { i } - y _ { i } \right|

    Euclidean distance¶

    Euclidean Distance adalah jarak dari dua buah titik dalam Euclidean space. Euclidean space diperkenalkan oleh Euclid, seorang matematikawan dari Yunani sekitar tahun 300 B.C.E. untuk mempelajari hubungan antara sudut dan jarak. Euclidean ini berkaitan dengan Teorema Phytagoras dan biasanya diterapkan pada 1, 2 dan 3 dimensi. Tapi juga sederhana jika diterapkan pada dimensi yang lebih tinggi.

    Average Distance¶

    Average Distance adalah versi modikfikasid dari jarak Euclidian untuk memperbaiki hasil.

    d _ { a v e } = \left ( \frac { 1 } { n } \sum _ { i = 1 } ^ { n } ( x _ { i } - y _ { i } ) ^ { 2 } \right) ^ { \frac { 1 } { 2 } }

    Weighted euclidean distance¶

    Weighted euclidean distance adalah modifikasi lain dari jarak Euclidean distance yang digunakan berdasarkan tingkatan penting dari atribut.

    d _ { w e } = \left ( \sum _ { i = 1 } ^ { n } w _ { i } ( x _ { i } - y _ { i } \right) ^ { 2 } ) ^ { \frac { 1 } { 2 } }

    Chord distance¶

    Chord distance adalah ukuran jarak modifikasi Euclidean distance untuk mengatasi kekurangan dari Euclidean distance yang akan dapat dipecahkan dengan menggunakan skala pengukuran.

    d _ { \text {chord} } = \left ( 2 - 2 \frac { \sum _ { i = 1 } ^ { n } x _ { i } y _ { i } } { \| x \| _ { 2 } \| y \| _ { 2 } } \right) ^ { \frac { 1 } { 2 } }

    Mahalanobis distance¶

    Mahabalanobis distance adalah jarak Mahalanobis dapat mengurangi distorsi yang disebabkan oleh korelasi linier antara fitur dengan menerapkan tranformasi pemutihan ke data atau dengan menggunakan kuadrat jarak Mahalanobis.

    d _ { m a h } = \sqrt { ( x - y ) S ^ { - 1 } ( x - y ) ^ { T } }

    Cosine measure¶

    Cosine Measure adalah ukuran Cosine Similarity yang lebih banyak digunakan dalam Simularitas dokumen.

    Cosine(x,y)=\frac { \sum _ { i = 1 } ^ { n } x _ { i } y _ { i } } { \| x \| _ { 2 } \| y \| _ { 2 } }

    Pearson correlation¶

    Pearson correlation adalah ukuran menghitung similaritis anatara duan bentuk pola expresi gen.

    Pearson ( x , y ) = \frac { \sum _ { i = 1 } ^ { n } ( x _ { i } - \mu _ { x } ) ( y _ { i } - \mu _ { y } ) } { \sqrt { \sum _ { i = 1 } ^ { n } ( x _ { i } - y _ { i } ) ^ { 2 } } \sqrt { \sum _ { i = 1 } ^ { n } ( x _ { i } - y _ { i } ) ^ { 2 } } }

    Mengukur Jarak Atribut Binary¶

    Mari kita lihat similaritas dan desimilirity untuk objek yang dijelaskan oleh atribut biner simetris atau asimetris. Aatribut biner hanya memiliki dua status: 0 dan 1 Contoh atribut perokok menggambarkan seorang pasien, misalnya, 1 menunjukkan bahwa pasien merokok, sedangkan 0 menunjukkan pasien tidak merokok. Memperlakukan atribut biner sebagai atribut numerik tidak diperkenankan. Oleh karena itu, metode khusus untuk data biner diperlukan untuk membedakan komputasi. $$ d ( i , j ) = \frac { r + s } { q + r + s + t } $$

    Mengukur Jarak Tipe categorical¶

    Overlay Metric¶

    Over Metric adalah mengukukur jarak masing-masing sample dengan melihat perbedaannya, karena gagal mendapatkan informasi oleh atribut nominal yang bisa membantu dalam generalisasi.

    d ( x , y ) = \sum _ { i = 1 } ^ { n } \delta ( a _ { i } ( x ) , a _ { i } ( y ) )

    Value Difference Metric (VDM)¶

    Value Difference Metric (VDM) adalah dua nilai dari nilai atribut yaitu lebih dekat jika memiliki klasifikasi yang sama. VDN dikenalkan oleh Standfill and Waltz, versi sederhana dari VDM tanpa skema pembobotan.

    d ( x , y ) = \sum _ { i = 1 } ^ { n } \sum _ { c = 1 } ^ { C } \left| P ( c | a _ { i } ( x ) ) - P ( c | a _ { i } ( y ) ) \right |

    Minimum Risk Metric (MRM)¶

    Minimun Risk Metric (MRM) adalah Meminimumkan selisih antara kesalahan sehingga dan kesalahan asymtotic yaitu meminimumkan risk of misclassification.

    d ( x , y ) = \sum _ { c = 1 } ^ { C } P ( c | x ) ( 1 - P ( c | y ) )

    Mengukur Jarak Tipe Ordinal¶

    Nilai-nilai atribut ordinal memiliki urutan atau peringkat, namun besarnya antara nilai-nilai berturut-turut tidak diketahui. Contohnya tingkatan kecil, sedang, besar untuk atribut ukuran. Atribut ordinal juga dapat diperoleh dari diskritisasi atribut numerik dengan membuat rentang nilai ke dalam sejumlah kategori tertentu. $$ z _ { i f } = \frac { r _ { i f } - 1 } { M _ { f } - 1 } $$

    Menghitung Jarak Tipe Campuran¶

    Menghitung ketidaksamaan antara objek dengan atribut campuran yang berupa nominal, biner simetris, biner asimetris, numerik, atau ordinal yang ada pada kebanyakan databasae dapat dinyatakan dengan memproses semua tipe atribut secara bersamaan. Salah satu teknik tersebut menggabungkan atribut yang berbeda ke dalam matriks ketidaksamaan tunggal dan menyatakannya dengan skala interval antar [0,0,1.0].

    d ( i , j ) = \frac { \sum _ { f = 1 } ^ { p } \delta _ { i j } ^ { ( f ) } d _ { i j } ^ { ( f ) } } { \sum _ { f = 1 } ^ { p } \delta _ { i j } ^ { ( f ) } }
