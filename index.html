    Statistika Deskriptif¶

    Rata-rata (Mean)¶

    Rata (mean) adalah sebuah rata-rata yang didpat dari hasil penjumlahan dari seluruh nilai masing-masing data, lalu akan dibagi dengan banyaknya data yang ada.

    Median¶

    Median adalah sebuah nilai tengah dari kumpulan data yang sudah diurutkan, jika jumlah data ganjil maka nilai median yaitu satu nilai yang berada ditengah urutan, tetapi jika jumlah data genap maka mediannya yaitu hasil penjumlahan dua nilai yang berbeda ditengah urutan data, lalu kemudia dibagi dua. $$ M_e = t_b + (\frac{\frac{1}{2}n-f_k}{f}) c $$

    = tepi bawah kelas median = banyak data = frekuensi kumulatif sebelum kelas median = frekuensi kelas median

    = panjang kelas

    Modus¶

    Modus adalah sebuah data atau nilai yang sering muncul atau yang memounyai jumlah frekuensi yang paling banyak.

    ​
    $$ mean − mode ≈3×( mean − median ) $$

    Standar Deviasi¶

    Standar Deviasi adalah sebuah nilai statistika yang dimanfaatkan untuk menentukan bagaimana sebaran data dalam sempel, serta seberapa dekat titik data individu ke mean ataupun rata-rata nilai sempel. $$ σ2=1NN∑i=1(xi−¯¯¯x)2=(1NN∑i=1x2i)−¯¯¯x2 $$

    Varian¶

    Varian adalah ragam atau distribusi probabilitas ukuran seberapa jauh sebuah kumpulan bilangan tersebut. varian nol mengindikasikan bahwa semua nilai sama. varian selalu bernilai non-negatif. varian yang rendah mengindikasikan bahwa titik data condong sangat dekat dengan nilai rerata dan antara satu sama lain.

    ​ $$ \sigma = \frac { 1 } { N } \sum _ { i = 1 } ^ { N } ( x _ { i } - \overline { x } ) ^ { 2 } = ( \frac { 1 } { N } \sum _ { i = 1 } ^ { N } x _ { i } ^ { 2 } ) - \overline { x } ^ { 2 } $$ ​

    Skewness¶

    Kemiringan (skewness) adalah tingkat ketidaksimestrisan atau kejauhan simetri dari sebuah distribusi, dan sebuah distribusi yang tidak simetris akan memiliki rata-rata, median, dan modus yang tidak sama besarnya sehingga distribusi akan terkonsentrasi pada salah satu sisi dan kurvanya akan menceng. $$ s k=\frac{\overline{X}-M o}{s} $$ Dengan $$ \overline{X}-M o \approx 3(\overline{X}-M e) $$ Maka $$ s k \approx \frac{3(\overline{X}-M e)}{s} $$

    Quartile¶

    Quartile adalah nilai-nilai yang membagi data dalam 4 bagian yang sama, atau setiap bagian dari kuartil sebesar 25%. Quartile itu ada 3, yaitu quartile pertama, kedua, dan ketiga. Adapun cara mencarinya hampir sama dengan cara mencari median, perbedaanya pada letak quartile. $$ I Q R = Q _ { 3 } - Q _ { 1 } $$
